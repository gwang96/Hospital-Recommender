# Polo-Care

## Usage
Query this server at /api?drg=###.
This should return a list of JSON Objects that have the fields
	- Provider Name
	- Provider Street Address
	- Provider City
	- Provider State
	- Provider Zip Code
	- Total Discharges
	- Average Total Payments
